# pihole-hosts
Nightly aggregated host blocklist compiled via GitHub Actions

* Built from data curated by [The Firebog](https://firebog.net/), using the ticked list found [here](https://v.firebog.net/hosts/lists.php?type=tick).
* Updated nightly.
